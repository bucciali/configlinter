# ConfigLinter

Утилита на Go для анализа конфигурационных файлов веб-приложений на предмет потенциально опасных настроек.

## Возможности

- **Поддержка форматов:** YAML, JSON, TOML
- **Правила безопасности:**
  - 🔴 `HIGH` — пароль в открытом виде
  - 🔴 `HIGH` — отключённый TLS
  - 🔴 `HIGH` — слабые/устаревшие алгоритмы (MD5, SHA1, DES, RC4)
  - 🟡 `MEDIUM` — привязка к `0.0.0.0` без ограничений
  - 🟢 `LOW` — логирование в debug-режиме
- **Режимы работы:**
  - CLI — проверка файла или папки (рекурсивно)
  - REST API (HTTP-сервер)
  - gRPC-сервер
- **Расширяемость:** новые правила, форматы и репортеры добавляются через интерфейсы
- **Проверка прав доступа** к файлам через `os.Stat`

## Установка
### Локально
go build -o configlinter ./cmd/main.go
### Docker
docker-compose up --build
